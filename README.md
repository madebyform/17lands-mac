# MTGA Log parser

Simple client for passing relevant events from MTG Arena logs to the 17Lands REST endpoint.

## For developers

### Installing dependencies

```bash
uv sync
```

### Running the log parser

Running using `uv`:

```
uv run 17lands.py
```
